Wasm Debugging Stories

List of core Wasm debugging user stories that must be supported appropriately by DevTools.

  • Wasm modules show up when opening DevTools
  • Use Response.url in DevTools
  • Raw Wasm module display with byte code offsets
  • Stepping from JavaScript into Wasm
  • Stepping out of Wasm and back into JavaScript
  • Call Stack when breaking in JavaScript from Wasm
  • Pause on exception with Wasm
  • Worker stuck in Atomics.wait
  • Missing indicator for worker status
  • Resume a worker after it has been paused in the debugger
  • Files in multiple workers
  • Break on script in multiple workers
  • Scopes
